# KaltraSounds

KaltraSounds is an all-in-one Paper sound platform intended to replace PlayMoreSounds on Kaltra RPG while preserving its public configuration style and useful addon behavior.

## Current candidate

- Version: `1.0.0-beta.7`
- Paper: `26.1.1`
- Java: `25`
- Added configurable region and biome loop timing, play limits, replay-volume fade-in, and optional replay-volume fade-out.

The source is reconstructed from the checksum-verified beta.6 source archive plus the checksum-verified beta.7 patch fragments. GitHub Actions compiles it against the real Paper API and boots a disposable Paper server for validation and reload smoke tests.
